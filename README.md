# Surakshith — Portfolio

A modern, animated personal portfolio built using **Next.js**, **Tailwind CSS**, and **Framer Motion**.  
It showcases my projects, skills, and contact information with clean UI, smooth micro-interactions, and a professional theme.

---

## 🚀 Live Website

🔗 **https://surakshith-portfolio.vercel.app/**

---

## 🛠 Tech Stack

- **Next.js**
- **React**
- **Tailwind CSS**
- **Framer Motion**
- **TypeScript**
- **Deployed on Vercel**

---

## ✨ Features

- Home, Projects, Skills & Contact sections
- Glass-morphism design with purple neon aesthetic
- Smooth animations using Framer Motion
- Responsive layout (mobile & desktop)
- Contact form powered by Formspree
- Fully deployable on Vercel
- SEO-friendly & fast
